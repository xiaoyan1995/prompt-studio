1. 软件组成

Prompt Studio Desktop 用于管理图片、视频、Skills 提示词、文档、音效和本地媒体资产。所有主要操作都在桌面端内完成。

组件作用是否必须
桌面端项目、提示词、文档、音效、设置和备份的主程序。必须
浏览器插件从网页采集图片/视频、反推提示词、快速插入提示词。可选
Skills让支持 Skill 的 AI Agent 读写 Prompt Studio 内容。可选
组件之间的关系 插件和 Skills 都依赖桌面端提供的本地服务。使用它们前，请先启动 Prompt Studio Desktop。

2. 安装与启动

安装版

  1. 双击 Prompt Studio Desktop Setup 1.2.2.exe。
  2. 按安装向导选择安装位置并完成安装。
  3. 从桌面或开始菜单启动 Prompt Studio Desktop。

绿色免安装版

先完整解压 ZIP，不要直接在压缩包内运行。正常目录结构如下：

Prompt Studio Desktop-1.2.2-portable-win\
├─ desktop\
│  ├─ Prompt Studio Desktop.exe    ← 双击启动
│  ├─ resources\
│  └─ studio-data\                 ← 用户数据
├─ extension\                           ← 浏览器插件
└─ skills\
   └─ prompt-studio\                    ← Agent Skill
不要只复制 EXE 绿色版必须保留 desktop 目录内的全部文件。插件和 Skills 可选，但使用时也要保留完整文件夹。

首次启动

  • 等待本地服务首次启动通常需要几秒。
  • 处理旧数据如出现数据迁移提示，可选择旧版 studio-data 或创建新数据。
  • 创建项目进入“提示词管理”，创建项目后开始导入内容。

3. 提示词管理

左侧导航会根据当前项目显示图片提示词、视频提示词、Skills 提示词、文档库和音效库。

图片与视频

  • 将图片/视频拖入软件，或粘贴剪贴板中的图片，会直接导入为空提示词卡片。
  • 导入时不弹参数框；需要补充标题、提示词、标签或参考图时，再点击卡片编辑。
  • 多选后可执行导出、批量反推等操作。

Skills 提示词

支持 Markdown 编辑和预览，也可导入多文件 Skill 文件夹。Skills 类型不执行图片/视频反推。

文档库与音效库

  • 文档库：用于保存和预览 PDF、Word、Excel、PowerPoint、TXT、Markdown 等文件。
  • 音效库：链接本地音频文件夹，可播放、搜索、收藏、多选并拖出到文件管理器或 DAW。

4. 资产浏览器

资产浏览器是一个独立板块，用于查看本地目录中的图片、视频和音频，不会自动把它们变成提示词卡片。

  1. 进入“资产浏览器”，点击“添加监控目录”。
  2. 选择本地文件夹，填写名称和需要扫描的子目录。
  3. 打开目录后，可通过左侧子目录、类型、分类和搜索筛选资产。
  • 单击选中，Ctrl + 单击追加多选。
  • Shift + 单击连续选择。
  • 在空白区域拖动鼠标可框选。
  • 可将本地文件拖入当前目录。
  • 可将已选资产拖出到本地。
  • 目录有新文件进入时会更新，也可手动刷新。
取消监控不会删除文件 “取消监控”只会从资产浏览器中移除目录记录，不会删除本地目录或其中的任何资产。

5. 浏览器插件

安装插件

  1. Chrome 打开 chrome://extensions/；Edge 打开 edge://extensions/。
  2. 开启“开发者模式”。
  3. 点击“加载已解压的扩展程序”。
  4. 选择包含 manifest.json 的 extension 文件夹。
  5. 将 Prompt Studio Desktop Companion 固定到浏览器工具栏。
版本插件目录
绿色版压缩包解压目录下的 extension
安装版<安装目录>\resources\extension

使用插件

  • 桌面端运行时，插件会连接 http://127.0.0.1:8768。
  • 鼠标悬停图片/视频可显示快捷工具栏；也可右键素材进行保存或反推。
  • 插件弹窗可批量扫描当前网页图片，并查看检测到的视频。
  • 对需要快速插入提示词的网站，在插件弹窗中启用该站点的插入开关。

禁止当前网页识别

  1. 在目标网页上打开插件弹窗。
  2. 点击顶部状态栏中的屏蔽图标；图标变为已屏蔽状态。
  3. 设置会保存在浏览器中，关闭并重开浏览器后仍然生效。
  4. 再点一次可取消屏蔽。也可在“插件设置 → 站点黑名单”中编辑，每行一个域名。
更新插件后必须重新加载 替换插件文件后，请到扩展程序管理页点击“重新加载”。如果直接换了插件文件夹路径，浏览器可能把它视为新插件，黑名单等设置需要重新配置。

6. 设置与 AI 功能

桌面端右上角“设置”中包含图片反推、视频反推、文本 LLM、FFmpeg、语言和数据管理。

  • 图片/视频反推：需配置对应的 API Base、API Key 和模型。
  • 文本 LLM：主要用于音效名翻译等文本功能。
  • FFmpeg：部分视频下载、合并和处理功能需要 FFmpeg。
  • 界面：支持中/英文和深色/浅色主题。
API Key 安全 API Key 会跟随桌面端设置保存。不要将自己的 studio-data、设置文件或已配置的绿色版整包发给其他人。

7. Skills 使用

skills\prompt-studio 是随项目提供的 Agent Skill，不是桌面软件必需文件，也不会自动修改你的全局 Skills。

  1. 按你使用的 AI Agent 平台规则，导入完整的 prompt-studio 文件夹。
  2. 启动 Prompt Studio Desktop。
  3. 使用明确指令调用，例如：
  • “用 Prompt Studio 列出我的项目。”
  • “把这段提示词保存到 Prompt Studio 的「项目名」。”
  • “查看资产浏览器中「监控目录名」的图片。”

成功时，Agent 会返回软件中真实的项目、提示词或资产结果。完整 API 约定见 skills\prompt-studio\REFERENCE.md。

8. 数据、备份与迁移

默认情况下，数据保存在主程序同目录的 studio-data 中。如果启动时选择了其他数据目录，以实际选择的位置为准。

内容建议
日常备份在“设置 → 运维”中创建本地快照。
整机迁移先退出软件，再复制完整的 studio-data 文件夹。
升级版本升级前先备份；绿色版不要直接覆盖唯一的数据副本。
对外分享仅分享干净程序包，不要携带个人 studio-data 和 API Key。

9. 常见问题

现象处理方法
双击后没有窗口等待几秒；如仍无窗口，在任务管理器中结束旧的 Prompt Studio Desktop 进程后重开。
绿色版无法启动确认已完整解压，并且 desktop\resources 与 EXE 仍在同一目录中。
插件显示未连接先启动桌面端；确认插件地址为 http://127.0.0.1:8768，再点击插件中的刷新。
插件更新后界面/屏蔽行为仍是旧的打开扩展程序管理页，对 Prompt Studio Desktop Companion 点击“重新加载”。
资产浏览器没有内容先添加监控目录，确认目录中有可识别的图片/视频/音频，然后点击刷新。
AI 反推失败在“设置 → API 配置”核对 API Base、Key 和模型；不要把完整提交接口重复填到 API Base。
反馈问题时请提供 软件版本、安装版/绿色版、浏览器名称、问题截图和复现步骤。请勿截图或发送 API Key。
